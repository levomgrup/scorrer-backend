Scorrer Backend
================

Amaç: TFF Süper Lig verilerini [TFF Lig Sayfası](https://www.tff.org/default.aspx?pageID=198) üzerinden çekip normalize ederek MongoDB Atlas'a yazmak.

Hızlı Komutlar
- Dev: `npm run dev`
- Build: `npm run build`
- Start: `npm start`

Klasör Yapısı (özet)
```
src/
  config/
  domain/
  application/
  infrastructure/
  interfaces/
  shared/
```

İlk Sprint: Süper Lig takımlarını çekip MongoDB'ye yazma.
